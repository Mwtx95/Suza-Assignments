Bootstrap Q&A

Why use Bootstrap?

Bootstrap offers several advantages:

  • Responsive design out of the box
  • Consistent UI components
  • Comprehensive documentation
  • Grid system for easy layouts
  • Extensive pre-built components
  • Community support
  • Cross-browser compatibility
  • Time-saving development

Difference between container, container-fluid, and row

Container: Has a fixed width that varies at different breakpoints. Centers content on the page with padding on both sides.

Container-fluid: Spans the entire width of the viewport (100% width) with padding on both sides. It's useful when you want full-width content.

Row: Used inside containers to create horizontal groups of columns. A row has negative margins (-15px by default) to offset the padding of containers, ensuring alignment. Rows must be placed inside containers, and columns must be placed inside rows.
