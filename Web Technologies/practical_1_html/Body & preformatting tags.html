HTML <body> and <pre> Tags Example

Demonstration:

This is preformatted text.
    It preserves spaces and line breaks.
        Useful for displaying code or poetry.g

Explanation:

<body>: The body tag defines the main content of the HTML document. It is where all visible elements (text, images, links, etc.) are placed. Here, we also used the background attribute of the <body> tag to set a background image.

<pre>: The pre tag displays preformatted text. Text inside <pre> is displayed in a fixed-width font, and whitespace (like spaces and line breaks) is preserved.
